El usuario de login **no viene de una migración ni de un seed**. Se crea automáticamente al arrancar el auth API.

## Origen del usuario admin

1. **Variables de entorno** (`.env` de ponti-auth):
   - `ADMIN_USER` → usuario
   - `ADMIN_PASS` → contraseña

2. **Creación automática** en `internal/platform/repository/pg/connection.go`:

```69:113:ponti-auth/internal/platform/repository/pg/connection.go
func createAdminUser(db *gorm.DB) error {
	// ...
	adminUsername := os.Getenv("ADMIN_USER")
	adminPass := os.Getenv("ADMIN_PASS")
	// Si no existe el usuario, lo crea con bcrypt
	result := db.Where("username = ?", adminUsername).First(&user)
	if result.Error == gorm.ErrRecordNotFound {
		admin := User{
			Username:   adminUsername,
			Password:   hashedPassword,
			IDRol:      1,  // admin
			// ...
		}
		db.Create(&admin)
	}
}
```

3. **Tabla `users`**: definida en `migrations/create_users_table.sql` (solo estructura, sin datos).
